bSecure for GiveWP

== Description ==

Introducing the latest way to make donations easier and more secure! With the bSecure plugin for GiveWP.

As the bSecure's advanced security features, you can rest assured that all donations made through your GiveWP form are safe and secure. This means you can focus on what really matters - raising money for your cause!

Switch to bSecure today and experience the convenience and security of GiveWP with the added protection of bSecure. Whether you're a non-profit organization, a charity, or an individual looking to raise funds for a good cause, bSecure's plugin for GiveWP is the perfect solution for you

== Installation ==

The easiest way to install bSecure for GiveWP plugin is to search for it on your site’s Dashboard.

STEP-BY-STEP INSTRUCTIONS

Log in to your site’s dashboard (e.g. www.yourdomain.com/wp-admin).
Click on the “Plugins” tab in the left panel, then click “Add New”.
Search for “bSecure for GiveWP” and find the plugin at the top.
Install it by clicking the “Install Now” link.
When installation finishes, click “Activate Plugin”.
Go to the settings section of bSecure within the GiveWP Payment Gateways tab
Add in the API Key & Credentials from your [bSecure account](https://partner.bsecure.pk/integration-live)
